### 使用到的融云技术

- 使用的是sealtalk的后端代码

### 项目运行

- 首先需要搭建好react native运行环境，具体可以参考[react native官方文档环境搭建](https://reactnative.dev/docs/environment-setup)

- 然后在项目根目录下运行 `yarn install`

- 连接好真机后，运行 `yarn android`
